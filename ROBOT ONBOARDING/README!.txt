ВНИМАНИЕ! На момент загрузки проекта, все находящиеся в нем тесты работают! По прохождению времени буду стараться их проверять и в случае присутствия ошибок исправлять.

Данный проект предназначен для автоматизированного тестирования онбординга в мобильном приложении "Афиша".
Проект состоит из простых базовых 4 тест-кейсов. Описание находится в самих тестах.

Все конфиги указаны в core>data>launcher.resource
